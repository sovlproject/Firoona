F.A.Q.

How do I import an animation? +

Open Firona and go to the Projects screen. Tap Import Project, then select a Firona project file from Files or iCloud Drive. The project will be added with all animation frames and timing preserved.

Which devices are supported? +

Firona is optimized for iPhone and iPad. Apple Pencil is highly recommended for professional frame-by-frame drawing.

What export formats are supported? +

Export your work as high-quality MP4 Video, Animated GIFs, individual frames as PNG, or a Printable Flipbook (PDF).
